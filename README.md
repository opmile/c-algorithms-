# C Language Exercises - Extensão do ip-studies
Esse repositório é uma extensão prática do meu outro repo [ip-studies](https://github.com/opmile/ip-studies), onde eu guardo toda a teoria da linguagem C e alguns exercícios. Aqui o foco é diferente: mão na massa, com resolução de vários exercícios para fixar conceitos importantes da linguagem.

## Sobre o repo
* Foi criado para me preparar para a prova de domínio extraordinário da disciplina Introdução à Programação da UFG.
* Contém exercícios focados em ponteiros, structs, alocação dinâmica de memória e conceitos fundamentais.
* Você não vai encontrar algoritmos complexos, grafos ou estruturas avançadas, auqi o foco é lógica de programação básica e na implementação correta.
* Muitos exercícios podem parecer repetidos, e foi intencional, porque a ideia era revisar escrevendo tudo de novo para fixar bem.

## Organização
O conteúdo está dividido em pastas temáticas para facilitar a navegação e a prática
* `array` - exercícios de arrays e manipulação de strings
* `logic` - exercícios para praticar lógica de programação bruta
* `memory` - exercícios de alocação e realocação dinâmica
* `recursion`- problemas resolvidos usando recursão
* `techniques` - explicação de técnicas de reslução de problemas (ex. programação dinâmica)
* `structs` - 
    * `group-by-data` - pequenos sistemas que relacionam a composição de structs, noções básicas de modelagem de dados (agrupamento por categoria)
* `systems` - pequenos sistemas, CRUDs e projetos que envolvem vários conceitos
